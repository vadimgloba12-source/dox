# OSINT Dox Bot (Telegram)

Telegram OSINT-бот с поиском по IP, телефону, ФИО, фотографиям, адресам и компромату. Работает через Google Search (searchapi.io), системой авторизации по номеру, внутренней валютой (звёзды) и полноценной админ-панелью.

## Возможности

| Функция | Стоимость | Описание |
|---|---|---|
| 🌐 **Поиск по IP** | 5 ⭐ | Геолокация, провайдер, координаты, карта |
| 📞 **Поиск по телефону** | 10 ⭐ | Оператор, страна + поиск владельца через Google |
| 👤 **Поиск по ФИО** | 15 ⭐ | Биография, деловые базы (rusprofile, ЕГРЮЛ) |
| 📸 **Фото человека** | 10 ⭐ | Фотографии через Google Images |
| 🏠 **Поиск адреса** | 15 ⭐ | Адрес через Google + ФССП, ЕГРЮЛ, ГАС Правосудие |
| 🕵️ **Компромат** | 20 ⭐ | Новости, суды, долги, банкротство |

Дополнительно:
- **Whitelist система** — только подтверждённые пользователи
- **Внутренняя валюта (звёзды)** — покупка за реальные деньги через администратора
- **Админ-панель** — подтверждение пользователей, начисление звёзд, статистика
- **Карты** — автоматически отправляет карту местности при поиске по IP

## Установка и запуск

### 1. Установка зависимостей

```bash
git clone <репозиторий>
cd dox-bot
npm install
```

### 2. Настройка `.env`

Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_telegram_id
SEARCHAPI_KEY=3U2BbwQzCxKvRzeaAATjeRz6
NUMVERIFY_API_KEY=demo
PORT=3000
```

| Переменная | Где получить |
|---|---|
| `BOT_TOKEN` | [@BotFather](https://t.me/BotFather) |
| `ADMIN_ID` | [@userinfobot](https://t.me/userinfobot) |
| `SEARCHAPI_KEY` | [searchapi.io](https://www.searchapi.io) |
| `NUMVERIFY_API_KEY` | [numverify.com](https://numverify.com) (опционально) |

### 3. Запуск

```bash
npm start
# или в режиме разработки:
npm run dev
```

## Использование

### Для пользователя

1. `/start` — регистрация (отправить номер телефона)
2. Дождаться подтверждения администратора
3. Выбрать действие в меню:
   - **🌐 Поиск по IP** — ввести IP-адрес → геолокация + карта
   - **📞 Поиск по телефону** — ввести номер → оператор + поиск владельца
   - **👤 Поиск по ФИО** — ввести имя → биография, деловые базы
   - **📸 Фото человека** — ввести имя → фотографии из Google Images
   - **🏠 Поиск адреса** — ввести имя/запрос → адрес из открытых источников
   - **🕵️ Компромат** — ввести имя → новости, суды, долги, банкротство
   - **⭐ Купить звёзды** — пополнить баланс

### Для администратора

```
/admin              — открыть панель администратора
/allow_user <id>    — подтвердить пользователя
/add_stars <id> <n> — начислить звёзды
/list_pending       — список ожидающих
/stats              — статистика
```

Первый администратор добавляется автоматически при запуске из `ADMIN_ID`.

## База данных

SQLite, файл `database.sqlite` создаётся автоматически.

| Таблица | Содержимое |
|---|---|
| `users` | telegram_id, phone, stars, allowed |
| `admins` | telegram_id, is_super |
| `requests` | user_id, type, query, cost, created_at |

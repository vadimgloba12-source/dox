# Dox Bot (Telegram версия)

Telegram бот для получения информации по IP‑адресам и номерам телефонов с системой авторизации по номеру, внутренней валютой (звёзды) и админкой.

## Возможности

- **Регистрация по номеру телефона** — пользователь отправляет контакт, администратор подтверждает.
- **Whitelist система** — только подтверждённые номера могут использовать бота.
- **Поиск по IP** — геолокация, провайдер, координаты (через ip‑api.com).
- **Поиск по номеру телефона** — валидация, оператор, регион (через numverify.com).
- **Внутренняя валюта (звёзды)** — запросы оплачиваются звёздами.
- **Покупка звёзд** — пакеты за реальные деньги (интеграция с администратором).
- **Админ‑панель** — подтверждение пользователей, начисление звёзд, статистика.
- **История запросов** — ведётся в базе данных.

## Установка и запуск

### 1. Клонирование и настройка

```bash
git clone <репозиторий>
cd dox-bot
npm install
```

### 2. Настройка переменных окружения

Создайте файл `.env` (уже есть пример) и укажите:

```env
BOT_TOKEN=your_telegram_bot_token_here
ADMIN_ID=your_telegram_id_here
NUMVERIFY_API_KEY=demo  # можно оставить demo, но лучше получить свой на numverify.com
PORT=3000
```

- **BOT_TOKEN** — получите у [@BotFather](https://t.me/BotFather).
- **ADMIN_ID** — ваш Telegram ID (можно узнать у бота @userinfobot).
- **NUMVERIFY_API_KEY** — ключ от [numverify.com](https://numverify.com) (бесплатный тариф: 100 запросов/месяц).

### 3. Запуск бота

```bash
npm start
```

Или в режиме разработки с nodemon:

```bash
npm run dev
```

Бот запустится и будет готов к работе.

## Использование

### Для пользователя

1. Начать диалог с ботом, отправив `/start`.
2. Отправить свой контакт (кнопка «📱 Отправить номер телефона»).
3. Дождаться подтверждения администратора.
4. После подтверждения использовать меню:
   - **🔍 Поиск по IP** — ввести IP‑адрес (стоимость: 5 ⭐).
   - **📞 Поиск по номеру** — ввести номер телефона (стоимость: 10 ⭐).
   - **⭐ Купить звёзды** — выбрать пакет и оплатить администратору.
   - **📊 Мой баланс** — проверить текущее количество звёзд.

### Для администратора

Команды:
- `/admin` — открыть панель админа.
- `/allow_user <id>` — подтвердить пользователя по его Telegram ID.
- `/add_stars <id> <количество>` — начислить звёзды.
- `/list_pending` — список ожидающих подтверждения.
- `/stats` — статистика бота.

Первый администратор добавляется автоматически при запуске, если в таблице `admins` пусто (берётся из `ADMIN_ID` в `.env`).

## База данных

Используется SQLite, файл `database.sqlite` создаётся автоматически. Таблицы:
- `users` — данные пользователей.
- `admins` — администраторы.
- `requests` — история запросов.

## Веб‑версия (опционально)

Если нужен веб‑интерфейс, можно запустить сервер Express:

```bash
npm run web
```

Сервер будет доступен на `http://localhost:3000` с теми же функциями, что и в Telegram боте.

# OSINT Dox Bot

Профессиональный Telegram OSINT-бот на базе Google Search API.  
Поиск по IP, телефону, ФИО, фото, адресу, email, нику, WHOIS и сбор компромата.

---

## Возможности

| Кнопка | Стоимость | Описание |
|---|---|---|
| 📋 **Полное досье** | 45 ⭐ | ФИО + фото + адрес + компромат одним запросом |
| 🌐 **IP-адрес** | 5 ⭐ | Геолокация, провайдер, тип (VPN/прокси/мобильный), карта |
| 📞 **Телефон** | 10 ⭐ | Оператор, страна, поиск владельца + профили в соцсетях |
| 👤 **ФИО** | 15 ⭐ | Биография (Google KG), соцсети (VK/OK/TG/Instagram/LinkedIn), деловые базы |
| 📸 **Фото** | 10 ⭐ | До 6 фотографий из Google Images (надёжная загрузка через буфер) |
| 🏠 **Адрес** | 15 ⭐ | Поиск адреса + извлечение из сниппетов, ФССП, ЕГРЮЛ, арбитраж |
| 🕵️ **Компромат** | 20 ⭐ | Новости/аресты, судебные дела, долги/банкротство, чёрные списки |
| 📧 **Email** | 10 ⭐ | Упоминания, связанные аккаунты, утечки, ссылки на HIBP |
| 👾 **Ник** | 10 ⭐ | Поиск по 14+ платформам: VK, TG, Instagram, TikTok, GitHub и др. |
| 🔍 **WHOIS/Домен** | 5 ⭐ | RDAP-данные: даты, регистратор, владелец, NS |

**Дополнительно:**
- 📜 История запросов (последние 10)
- ⭐ Встроенная валюта — покупка через администратора
- 👥 Whitelist — только подтверждённые пользователи
- 🔒 Rate limiting — до 30 запросов в час на пользователя
- ✅ Валидация ввода (IP, телефон, email, домен)
- 🔘 Быстрые действия после ФИО-поиска: фото / компромат / адрес

## Быстрый старт

### 1. Установка

```bash
git clone https://github.com/vadimgloba12-source/dox
cd dox
npm install
```

### 2. Настройка

```bash
cp .env.example .env
```

Откройте `.env` и заполните:

```env
BOT_TOKEN=токен_от_BotFather
ADMIN_ID=ваш_telegram_id
SEARCHAPI_KEY=3U2BbwQzCxKvRzeaAATjeRz6
```

| Переменная | Где получить |
|---|---|
| `BOT_TOKEN` | [@BotFather](https://t.me/BotFather) → /newbot |
| `ADMIN_ID` | [@userinfobot](https://t.me/userinfobot) |
| `SEARCHAPI_KEY` | Уже вписан. Если кончатся запросы — [searchapi.io](https://searchapi.io) |

### 3. Запуск

```bash
npm start
```

Режим разработки (авторестарт):
```bash
npm run dev
```

## Команды

### Пользователь
| Команда | Действие |
|---|---|
| `/start` | Регистрация / главное меню |
| `/menu` | Открыть главное меню |
| `/balance` | Баланс звёзд |
| `/history` | История запросов |
| `/cancel` | Отменить текущее действие |

### Администратор
| Команда | Действие |
|---|---|
| `/admin` | Панель администратора |
| `/allow_user <id>` | Подтвердить пользователя |
| `/add_stars <id> <n>` | Начислить звёзды |
| `/ban <id>` | Заблокировать пользователя |
| `/unban <id>` | Разблокировать |
| `/broadcast <текст>` | Рассылка всем активным |
| `/list_pending` | Список ожидающих |
| `/stats` | Статистика |
